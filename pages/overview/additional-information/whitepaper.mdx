**Авторы:** Габриэле Пикко, Андреа Фортуньо 

Мы предлагаем фреймворк, который использует **Solana Virtual Machine** (SVM) для масштабирования полностью on-chain приложений без фрагментации состояния и без компромиссов в доверительных предположениях.

Для повышения масштабируемости и оптимизации ресурсов мы вводим концепцию **Эфемерных Роллапов (ERs):**

- Выделенные среды исполнения, настраиваемые для более высокой скорости работы.
- Настраиваемые механизмы «тиков» (циклов обработки).
- Доказуемые сессии и транзакции без комиссии.
- Обеспечивает масштабируемость без ущерба для компонуемости.
