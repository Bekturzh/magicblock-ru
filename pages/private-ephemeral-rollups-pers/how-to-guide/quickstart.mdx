## Пошаговое руководство

Создайте вашу программу и обновите её с хуками разрешений и делегирования, используя Permission Program MagicBlock `ACLseoPoyC3cBqoUtkbjZ4aDrkurZW86v19pXz2XQnp1` и Delegation Program `DELeGGvXpWV2fqJUhqcF5ZSYMS4JTLjteaAMARRSaeSh`:

<Steps>
  <Step title={<a href="#1-write-program">Пишите вашу программу</a>}>
    **Пишите вашу программу на Solana как обычно.**
  </Step>
  <Step title={<a href="#2-restrict">Добавьте ограничения с помощью перехватчиков разрешений в вашей программе.</a>}>
    Добавьте хуки разрешений для создания и управления контролем доступа, [посмотреть детали](/pages/private-ephemeral-rollups-pers/how-to-guide/access-control)
  </Step>
  <Step title={<a href="#3-delegate">Добавьте перехватчики делегирования в вашу программу</a>}>
    **Добавьте хуки делегирования для разрешений и аккаунтов с ограниченным доступом, чтобы применять ограничения.**

    <Validators />
  </Step>
  <Step title={<a href="#4-deploy">Разверните вашу программу на Solana</a>}>
    Разверните вашу программу на Solana с помощью Anchor CLI.
  </Step>
  <Step title={<a href="#5-authorize">Подпишите сообщение пользователя</a>}>
    Подпишите сообщение пользователя, чтобы получить токен авторизации с СДИ-эндпоинта.
  </Step>
  <Step title={<a href="#6-test">Выполняйте транзакции и проверяйте конфиденциальность</a>}>
    Запросите токен авторизации и отправляйте конфиденциальные транзакции.
  </Step>
</Steps>

## Пример «Камень-Ножницы-Бумага»

Могут потребоваться следующие пакеты программного обеспечения, другие версии также могут быть совместимы:

| **Программное обеспечение** | **Версия** | **Руководство по установке**                                         |
| --------------------------- | ---------- | -------------------------------------------------------------------- |
| **Solana**                  | 2.3.13     | [Установите Solana](https://docs.anza.xyz/cli/install)               |
| **Rust**                    | 1.85.0     | [Установите Rust](https://rust-lang.org/tools/install/)              |
| **Anchor**                  | 0.32.1     | [Установите Anchor](https://www.anchor-lang.com/docs/installation)   |
| **Node**                    | 24.10.0    | [**<u>Установите Node</u>**](https://nodejs.org/en/download/current) |

<Note>
  The latest Permission Program requires SDK version [\>=0.8.0](https://github.com/magicblock-labs/ephemeral-rollups-sdk). See [migration
  guide](https://github.com/magicblock-labs/ephemeral-rollups-sdk/pull/103) for details.
</Note>
