### Модель авторизации

<img
  class="w-full h-auto max-w-5xl"
  src="/images/TEE_permission.png"
/>

**Приватный Эфемерный Роллап используют Программа разрешений для управления детализированными настройками конфиденциальности для аккаунтов и групп аккаунтов. Она работает на Solana L1 и может обновляться на лету.**

- **Permission Groups:** Определяйте группы с произвольным составом участников и идентификаторами через CPI. Группа объединяет пользователей и аккаунты, управляемые её разрешениями.
- **Permissions:** Добавляйте разрешения к группам. На сегодняшний день разрешение подразумевает доступ только для чтения делегированного аккаунта; в будущем могут быть добавлены разделения на чтение/запись.
- **Access:** Доступ клиента к состоянию ЭР с разрешениями требует аутентификации владения указанным публичным ключом. Успешная аутентификация выдаёт токен, используемый для запросов к ЭР.
