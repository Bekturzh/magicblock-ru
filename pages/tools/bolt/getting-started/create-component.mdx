Компоненты — это простые структуры данных (классы или структуры), содержащие информацию, относящуюся к определённому аспекту сущности. Они не содержат логики или методов. Пример компонента — компонент «Position», который хранит координаты x, y, z.

Создайте компонент Position с

```bash
bolt component position
```

который добавит новый компонент в ваше рабочее пространство:

```rust
use bolt_lang::*;

declare_id!("Fn1JzzEdyb55fsyduWS94mYHizGhJZuhvjX6DVvrmGbQ");

#[component]
#[derive(Copy)]
pub struct Position {
    pub x: i64,
    pub y: i64,
    pub z: i64,
}
```
