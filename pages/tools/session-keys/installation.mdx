---

Сессионные ключи являются частью **Gum React-SDK**, поэтому необходимо установить пакет с помощью **yarn** или **npm**:

<CodeGroup>

```bash Yarn
yarn add @magicblock-labs/gum-react-sdk
```


```bash Npm
npm i @magicblock-labs/gum-react-sdk
```

</CodeGroup>

После установки пакета вы можете начать интеграцию системы управления сессионными кошельками в ваш dApp для повышения безопасности и улучшения пользовательского опыта.
