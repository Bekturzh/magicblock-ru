## **Solana.Unity-SDK**

**Solana.Unity SDK** — это комплексный набор open-source инструментов для лёгкого доступа к Solana в играх на базе Unity. Вы можете установить SDK через **Unity Package Manager** или в **Unity Asset Store**, а также настроить предпочитаемый кошелёк из доступных вариантов. **Solana.Unity-SDK** использует реализацию **Solana.Unity-Core**, нативную для .NET Standard 2.0 (совместимую с Unity), с полной поддержкой RPC API, MPL, нативными операциями DEX и другими возможностями.

**Функции:**

- Полная поддержка JSON RPC API
- Кошельки и аккаунты: настройка некостодиального кошелька Solana в Unity (совместим с sollet и solana-keygen)
- Поддержка Phantom и Web3auth (некостодиальная регистрация/вход через социальные аккаунты)
- Декодирование транзакций из base64 и wire-формата и обратное кодирование в wire-формат
- Декодирование сообщений из base64 и wire-формата и обратное кодирование в wire-формат
- Декомпиляция инструкций
- Объект **TokenWallet** для отправки и получения SPL токенов и JIT создание Associated Token Accounts
- Примеры базового UI
- Поддержка NFT
- Компиляция игр в xNFT (Backpack)
- Нативные операции DEX (Orca, Jupiter — скоро...)
- WebSockets для регистрации/триггера пользовательских событий (изменение аккаунта, статус подписи, программы и др.)
- Поддержка **Solana Mobile Stack**
- **Solana Wallet Adapter**
