## **Функции**

- **Prefabs** — это шаблоны, которые сохраняют конфигурацию объектов. Их можно использовать для создания экземпляров одних и тех же ресурсов, и они легко переиспользуются.
- Папка **Scenes** содержит полный демонстрационный пример, развернутый здесь.
- Папка **Scripts** содержит различные примеры скриптов (например, `Assets/Samples/Solana SDK/0.1.1/Sample Wallet/Solana Wallet/Scripts/example/screens/SwapScreen.cs` для Orca Swaps).
- **Animations**, **Materials**, **Plugins** и **Textures** объясняются сами собой.

Попробуйте поиграть с демонстрационной сценой, чтобы ознакомиться с базовой функциональностью: кошельки, загрузка NFT, переводы, свопы и др.
