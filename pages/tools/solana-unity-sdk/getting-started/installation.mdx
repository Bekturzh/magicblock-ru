## **Unity**

Перейдите  [по](https://unity.com/download) ссылке, чтобы установить Unity.

**Импорт пакета SDK**

1. Откройте окно **Unity Package Manager**.
2. Нажмите кнопку **\+** в строке состояния.
3. Появятся варианты добавления пакетов.
4. Выберите [Unity Package Manager](https://docs.unity3d.com/Manual/upm-ui.html) в меню добавления. Появятся текстовое поле и кнопка **Add**.
5. ![](/images/package_manager.png)
6. Введите Git URL [**https://github.com/magicblock-labs/Solana.Unity-SDK.git**](https://github.com/magicblock-labs/Solana.Unity-SDK.git) в текстовое поле и нажмите **Add**.

   После установки пакета в инспекторе **Package Manager** появятся **Samples**. Нажмите **Import**.

   Также можно установить конкретную версию пакета, используя URL с указанием версии:

   ```
   https://github.com/magicblock-labs/Solana.Unity-SDK.git#X.Y.X
   
   ```

   Обратите внимание, что **X.Y.X** следует заменить на нужную вам версию.

   Все доступные релизы можно найти [здесь](https://github.com/magicblock-labs/Solana.Unity-SDK/releases).

   Последняя доступная версия релиза — **[указать последнюю версию]**.

   **Импорт примера сцены**
7. ![](/images/sample_scene.png)
8. Вы найдёте пример сцены с настроенным кошельком по пути:

   **Samples/Solana SDK/0.0.2/Simple Wallet/Solana Wallet/scenes/wallet_scene.unity**
