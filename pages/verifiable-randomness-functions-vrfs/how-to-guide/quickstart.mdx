## **Пошаговое руководство**

Любая программа на Solana может запрашивать и использовать проверяемую случайность (verifiable randomness) в сети за считанные секунды с помощью MagicBlock VRF SDK. К концу этого руководства у вас будет рабочий пример, который бросает кубик с использованием проверяемой случайности.

<Steps>
  <Step title={<a href="#1-write-program">Напишите вашу программу</a>}>
    Пишите вашу программу на Solana как обычно.
  </Step>
  <Step title={<a href="#2-request-%26-consume-randomnness">Добавьте запрос и используйте инструкции по случайному использованию.</a>}>
    Добавьте CPI-хуки, которые запрашивают и используют случайность через обратный вызов от проверенного оракула.
  </Step>
  <Step title={<a href="#3-deploy">Разверни свою программу на Solana</a>}>
    Разверните вашу программу на Solana с помощью Anchor CLI.
  </Step>
  <Step title={<a href="#4-test">Выполняйте транзакции для обеспечения случайности в цепочке.</a>}>
    Отправляйте транзакции для генерации и использования случайности в сети.
  </Step>
</Steps>

  <Tab title="2. Запрос и использование случайности">
    1. Добавь `ephemeral_vrf_sdk` с использованием возможностей Anchor в вашей программе

    ```bash
    cargo add ephemeral_vrf_sdk --features anchor
    ```

    Импортируй`vrf` , `create_request_randomness_ix`, `RequestRandomnessParams`, и`SerializableAccountMeta`:

    ```rust
    use ephemeral_vrf_sdk::anchor::vrf;
    use ephemeral_vrf_sdk::instructions::{create_request_randomness_ix, RequestRandomnessParams};
    use ephemeral_vrf_sdk::types::SerializableAccountMeta;
    ```

    2. \*\*Добавьте инструкции \*\*`roll_dice`\*\*для запроса случайности и \*\* `callback_roll_dice`**для её использования, вместе с соответствующим контекстом:**

    > Использование случайности — это процесс применения проверяемой случайности вашей программой, предоставляемой и активируемой через проверенный оракул.
